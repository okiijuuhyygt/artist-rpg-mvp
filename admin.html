♪ 音檔上傳

MP3 / WAV / M4A — 上傳後取代預設 8-bit 音。空的槽用內建 8-bit。

iPhone 上傳語音備忘錄: Voice Memos App 選錄音 → 分享 → 「儲存到檔案」→ 回這裡按「選音檔」→ 選「選擇檔案」→ 找剛存的那個。 或去「設定 → Apple ID → iCloud → 語音備忘錄」開同步,之後 Files App 會有「語音備忘錄」資料夾直接選。
